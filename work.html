'Avatar: The Last Airbender' Terminal Game (C++)

This project is a text based, object oriencted, interactive 'Avatar: The Last Airbender' themed game. It tests the user's knowledge of the show through trivia based battles. The goal of the game is to progress through nations gaining elemental levels to ultimately beat the final boss: Lord Ozai.

'Familiar Apprentice' Pixel Game (HTML/CSS/Javascript)

This project is a pixel story game featuring a cat who is trying to find its witch, prompting an adventure through the forest. I created all of the art in Photoshop. The code is inspired by Drew Conley’s Youtube playlist for his game, “Pizza Legends - Javascript RPG” (www.youtube.com/@DrewConley)

'Culture Battle' Fighter Game (HTML/CSS/Javascript)

This project is a fighter game that highlights the duality in Japanese culture between the stereotypes that result from the idea of homogenous Japan and the marginalized, precariat individuals. This code is inspired by Chris Courses HTML Fighter Game (www.youtube.com/@ChrisCourses)
