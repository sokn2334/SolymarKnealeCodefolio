ABOUT THIS PROJECT
------------------------
This project is a text based interactive 'Avatar: The Last Airbender' themed game.
It tests the user's knowledge of the show through trivia based battles.
The goal of the game is to progress through nations gaining elemental levels to ultimately
beat the final boss: Lord Ozai.
